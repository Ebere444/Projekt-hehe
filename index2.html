Metódusok

Az eljárás

Az eljárás egy olyan metódus a C# programozási nyelvben, amely nem ad vissza értéket. Az eljárások célja általában egy adott művelet végrehajtása, például egy szöveg kiírása a konzolra vagy egy feladat elvégzése a programban.

Az eljárások visszatérési típusa void, ami azt jelenti, hogy a metódus nem ad vissza eredményt.

Az eljárások segítenek abban, hogy a programunk átláthatóbb és könnyebben kezelhető legyen, mivel egy gyakran használt műveletet külön metódusba szervezhetünk.

Példa eljárásra:

static void Koszones()
{
    Console.WriteLine("Szia!");
}

A metódus meghívása:

Koszones();

Ebben az esetben a program kiírja a konzolra a „Szia!” szöveget.

A függvény

A függvény egy olyan metódus, amely visszaad egy értéket. Ez azt jelenti, hogy a függvény elvégzi a feladatát, majd egy eredményt ad vissza a programnak.

A függvények visszatérési típusa lehet például:

  • int (Pozitív és negatív egész számok)
  • double (Pozitív és negatív valós számok)
  • string (Szöveg)
  • bool (Igaz és Hamis érték)

A visszatérési értéket a return kulcsszó segítségével adjuk vissza.

Példa függvényre:

int Osszeadas(int a, int b)
{
    return a + b;
}

A függvény használata:

int eredmeny = Osszeadas(3, 5);

Console.WriteLine(eredmeny);
